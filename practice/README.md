# practice
`utils.py` のクラスや関数を使って実際にできることの事例を紹介する.

## LocateTheaters
日本全国の TOHO シネマズの劇場の住所を緯度経度の情報に変換して地図上にプロットする.

## PaperFetch
論文の PDF をダウンロードする.

## StockCheck
日経平均株価のアーカイブデータを WEB サイトからスクレイピングして取ってくる.

## VisualizeDirectory
`utilizers/practice` 以下のディレクトリ構造を可視化する.

## YouTube
YouTube から動画をダウンロードする.
